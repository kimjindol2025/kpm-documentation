## 🆕 신규 기능

### 1. 태그별 검색
**추가됨**: 2026-02-25

```bash
$ kpm tag <태그명>
# 태그로 패키지 검색
# 예: kpm tag async, kpm tag ai
```

**설명**: 카테고리/기능 태그로 관련 패키지를 한 번에 검색
**장점**: 더 빠른 패키지 발견
**테스트**: ✅ 완료

---

### 2. 패키지 발행 (publish)
**추가됨**: 2026-02-26

```bash
$ kpm publish
# 현재 디렉토리의 package.json을 기반으로 레지스트리에 등록
```

**설명**: 새 패키지를 KPM 레지스트리에 자동 등록
**선행조건**: package.json 필수
**테스트**: ✅ 완료

---

### 3. npm → KPM 마이그레이션 분석
**추가됨**: 2026-02-27

```bash
$ kpm analyze package.json
# npm 프로젝트의 의존성을 KPM에서 대체 가능한지 분석
```

**설명**: npm에서 KPM으로 점진적 마이그레이션 지원
**장점**: 기존 npm 프로젝트의 KPM 호환성 검증
**테스트**: ✅ 완료
